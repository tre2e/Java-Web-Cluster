## 🚀 项目介绍：Java Web Cluster - 智能广告投放网站群

### 🎯 项目概述

本项目旨在构建一个**基于智能广告投放的 Java Web 网站集群**，通过连接一个核心**广告投放网站**与三个面向不同用户群体的垂直应用网站（**购物**、**视频**、**新闻**），形成一个完整的生态系统。它不仅是一个实验性作业，更是一个涵盖现代 Web 开发核心技术的综合性平台，目标是实现精准广告投放、高效内容管理和多站协同运营。



### 💻 核心技术栈

### 🌐 网站集群构成与功能详情

本项目由一个核心管理网站和三个用户端网站组成，共同实现广告的投放、展示和效果追踪。

#### 1. 📢 广告投放网站 (面向广告投放方/管理员)

* **核心功能：** 广告管理、效果追踪、内容分发。
* **管理员操作：**
    * 投放、编辑、管理**文字**、**图片**、**视频**三种形式的广告。
    * 设置广告的**偏好属性**（例如：目标用户、内容类型）。
    * 实时查看并分析各广告的**点击数**（Click-Through Rate, CTR）等投放效果指标。
* **智能投放：** 根据管理员设定的偏好和网站属性，将广告智能地分发到购物、视频、新闻三个网站上。

#### 2. 🛍️ 购物网站 (面向顾客群体)

* **核心功能：** 商品浏览、内容发布、广告展示。
* **用户体验：** 游客可以浏览丰富的商品信息。
* **管理员操作：** 负责上传和管理商品内容。
* **广告集成：** 在页面显眼位置展示来自广告网站的广告，实现内容与商业的结合。

#### 3. 📺 视频网站 (面向视频观众)

* **核心功能：** 视频内容聚合、播放、主页广告展示。
* **用户体验：** 用户可在主页看到多个视频缩略信息，点击后观看具体视频内容。
* **广告集成：** 在**主页面**的关键区域投放广告，获取高曝光率。

#### 4. 📰 新闻网站 (面向新闻读者)

* **核心功能：** 新闻分类浏览、内容阅读、侧边广告推送。
* **用户体验：** 用户可通过菜单选择不同类别的新闻进行查看。
* **广告集成：** 在新闻阅读页面的**两侧边栏**区域推送精准广告，确保不影响阅读主体的同时实现商业价值。

### 💡 项目价值与亮点

1.  **实战经验：** 全面掌握从前端到后端，从数据库到集群部署的 Java Web 全栈开发流程。
2.  **数据驱动：** 实现广告点击数据的收集和展示，体现数据分析在商业中的应用。
3.  **多站协同：** 核心在于构建一个跨越多个独立应用的广告分发和管理机制，模拟真实商业环境。
4.  **内容多样性：** 支持文字、图片、视频三种广告形式，体现良好的系统扩展性。
