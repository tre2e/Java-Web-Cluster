## 开发内容
1. **广告投放网站**
2. **购物网站**
3. **视频网站**
4. **新闻网站**
### 大致介绍
1. 广告网站面向广告投放方，实现管理员可以在上面投放并且管理广告，并且可以看到广告的点击数。根据偏好将广告投放到其他三个网站，广告分为文字、图片和视频三种形式。
2. 购物网站面向顾客群体，实现游客浏览商品信息，并且可以看到免费的广告，同时管理员负责发送购物内容与广告。
3. 视频网站面向看视频的群众，实现游客可以在主页面看到多个视频信息，点击后可以看到具体视频，并且在主页投放广告。
4. 新闻网站面向看新闻的群众，实现游客可以在菜单上选择新闻点击查看，并且两侧推送广告。
